# UI Spec

## Dashboard

- Domestic and US market filters.
- Label filter.
- Minimum score filter.
- Candidate table with final score, model probability, expected return, downside
  risk, confidence, reasons, risk flags, and source references.
- Market regime summary.
- Required disclaimer.

## Stock Detail

- Header with ticker, exchange, market, label, confidence, and risk level.
- Score breakdown.
- Reasons and risk flags.
- Price and volume summary.
- Backtest/validation context.
- Source references.

No UI copy may use trade directives, profit guarantees, or hype framing.
