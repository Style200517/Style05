# Image-Based Design Prompt Archive

아래 프롬프트는 생성된 SignalFlow 1920×1080 대시보드 이미지를 다시 만들거나 변형할 때 사용한다.

## Base Image Generation Prompt

Create a polished 1920x1080 desktop web UI design mockup for a Korean stock signal and investment analysis platform.

The visual language should strongly resemble a clean Korean exchange-style desktop dashboard with white background, thin gray separators, compact tables, dark gray typography, and orange primary accents.

Do not copy Bithumb’s brand name, logo, exact text, or exact proprietary layout. Create an original product named SignalFlow.

The product is a stock signal research dashboard, not a crypto exchange and not a direct investment recommendation app.

## Visual Style

Use:

- White background
- Orange accent
- Light gray card border
- Compact layout
- Small badges
- Sharp financial dashboard typography
- Thin table lines
- Production-ready web UI
- Front-facing screenshot
- No device frame
- No perspective distortion

Avoid:

- Neon
- Glassmorphism
- Heavy shadow
- Mobile app look
- 3D mockup
- Gaming UI
- Crypto orderbook focus
- Direct buy/sell recommendation copy

## Copy Rule

Use:

- 관찰
- 관망
- 주의
- 제외
- 상승 가능성 점수
- 판단 근거
- 리스크
- 신뢰도
- 위험도

Do not use:

- 매수
- 매도
- 수익 보장
- 급등 확정
- AI 추천주
